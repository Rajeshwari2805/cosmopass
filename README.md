# 🚀 COSMOPASS — Space Event Booking

A full stack ticket booking web app for space events.

## 🔗 Live Demo
[Click Here to View](https://cosmopass.infinityfreeapp.com/cosmopass/)

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Hosting: InfinityFree

## ✨ Features
- 3 space events with live seat availability
- Interactive VIP & Standard seat map
- Real-time booking with seat locking
- Admin dashboard with revenue stats
- Unique PASS ID generated per booking
